What's New!
Recent QUOREM Changes

October 8, 2021:

  • Representative sequences are now stored in the database and appear on the Feature detail page
  • Results pages for QIIME artifacts now show the command used to generate that artifact
  • Results pages now highlight the non-default parameters used
  • Changes to the backend so that ASV/OTU/Feature tables can now be output in Pandas DataFrame format, making export and visualization simpler
August 30, 2021:
  • Added 'What's New!' and 'Database Summary' sections to landing page (here!)
  • All object list pages now have a filter on the "Name" field
  • Feature List page now lists related Analyses and Taxonomic Classifications
  • Feature List can now filter by name, related analysis, and taxonomic classification
  • Sample List page now lists related Analyses and number of Features in each Sample
  • Result List page now lists related Analyses, type of Result in Human English (as opposed to QIIME2 English), uploader, and a link to the Artifact download, if possible
  • Analysis List page now lists the # of related Samples, Features, and Results
  • Provenance graphs on Result detail pages are now simplified
  • Download buttons are present on each Result detail page, and an Artifact/Visualization link will be present if a file exists
  • Analysis detail pages now list all Results, with Results in green being available for download and/or visualization
  • Visual tweaks for autofill dropdowns to avoid graphical glitches with various screen widths
